- **版本**: 1.0.0

## 打包方式
本 Mod 使用 ForgeGradle 进行构建，并采用 shadowJar 插件来打包，确保所有必要的依赖都被正确地包含在最终的 JAR 文件中。这样做是为了确保在 Minecraft 游戏中的兼容性和稳定性。
